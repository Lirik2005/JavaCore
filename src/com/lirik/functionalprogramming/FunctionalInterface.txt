Comparator <T> - используется для сравнения чего-либо. Принимает два объекта <T>, сравнивает их и возвращает int (-1, 0, 1):
 -1, если первый объект меньше второго объекта;
  0, если объекты равны;
  1, если первый объект больше второго.

Function <T, R> - принимает на вход объекта типа <T> и возвращает объект типа <R>

Predicate <T> - используется для фильтрации значений. Принимает на вход объект типа <T> и возвращает boolean

Consumer <T> - принимает на вход объект типа <T>, использует его и ничего не возвращает

Supplier <T> - ничего не принимает, но возвращает объект типа <T>

BiConsumer <T, U> - принимает <T> и <U> и ничего не возвращает
